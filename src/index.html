MinTIC inicia proyecto para formar 100.000 colombianos en lenguaje de programación

La ministra TIC, Karen Abudinen, sostuvo una nueva reunión con líderes de empresas y emprendedores para fortalecer el proyecto Jóvenes 4.0. Con esta iniciativa se busca cerrar la brecha de talento digital, que para 2025 se proyecta entre 68.000 y 112.000 desarrolladores de software.
